## Compile, Build & Package

*Requirements: Java >= 8, Maven >= 3.3.9*

~~~BASH
> git clone https://github.com/pbelmann/ObjectStorageWrapper.git
> cd ObjectStorageWrapper
> mvn clean package
~~~
